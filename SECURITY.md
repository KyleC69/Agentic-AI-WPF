# Security Policy

## Supported Versions

I will maintain any version and support updates for all released software.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |                |

## Reporting a Vulnerability

If you discover a security issue in my code please tell me about it and post in the issues section so it can be addressed immediately.

You can also email me directly at KCrowdergoog@gmail.com for any questions or comments.

Tell them where to go, how often they can expect to get an update on a
reported vulnerability, what to expect if the vulnerability is accepted or
declined, etc.
